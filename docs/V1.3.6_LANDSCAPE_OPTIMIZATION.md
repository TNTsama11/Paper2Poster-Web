# v1.3.6 横版模板优化

## 问题反馈

用户反馈：
1. **分辨率太高** (4K 3840×2160) - 内容被稀释
2. **图片少时空白太多** - 应增加文字内容
3. **Abstract 内容不全** - 显示被截断

---

## 优化方案

### 1. 降低默认分辨率 ✅

**修改前**: `screen_4k` (3840×2160)  
**修改后**: `screen_hd` (1920×1080)

**原因**:
- 4K 分辨率过高，内容密度低，显得稀疏
- 1080p 更符合实际使用场景（投影仪、在线会议）
- 内容密度更高，视觉效果更紧凑

**配置文件** (`config.py`):
```python
TEMPLATE_RECOMMENDED_SIZES = {
    "academic_panels_landscape.html": "screen_hd",  # 1920×1080
}
```

---

### 2. 增加内容密度 ✅

#### 2.1 布局优化

| 项目 | 修改前 | 修改后 | 说明 |
|------|--------|--------|------|
| **左侧栏宽度** | 35% | 32% | 给右侧更多空间 |
| **容器间距** | 20px | 15px | 减少空白 |
| **容器内边距** | 20px | 15px | 增加内容区 |
| **面板间距** | 12px | 10px | 更紧凑 |
| **面板内边距** | 16px | 14px | 增加内容空间 |

#### 2.2 字体优化

| **图片说明** | 0.78rem | 0.72rem | 减小 |

#### 2.3 图片尺寸优化

| 位置 | 修改前 | 修改后 | 减少 |
|------|--------|--------|------|
| **普通面板** | 350px | 280px | -20% |
| **主图面板** | 600px | 450px | -25% |
| **高面板** | 500px | 380px | -24% |

---

### 3. 修复 Abstract 显示 ✅

#### 问题分析

Abstract 内容不全的原因：
1. 容器高度不够
2. 字体太大
3. 行高过大

#### 解决方案

```css
.panel-abstract .panel-content {
    font-size: 0.85rem;        /* 减小字体 */
    line-height: 1.55;         /* 适中行高 */
    text-align: justify;       /* 两端对齐 */
    overflow-y: auto;          /* 允许滚动 */
    max-height: 400px;         /* 限制最大高度 */
}
```

**效果**:
- ✅ 更小的字体可以显示更多内容
- ✅ 如果内容过多，允许滚动查看
- ✅ 最大高度400px，留出主图空间

## 使用方法

### CLI
```bash
# 默认使用 1080p
python main.py paper.pdf -t academic_panels_landscape.html

# 手动指定 4K（如果需要）
python main.py paper.pdf -t academic_panels_landscape.html --preset screen_4k
```

### Web UI
选择：**学术会议-横向16:9 (academic_panels_landscape) 🆕**

系统会自动使用 **1920×1080** (1080p)

## 何时使用 4K？

如果您的论文有以下特点，可以手动选择 4K：
- ✅ 图片特别多 (>15张)
- ✅ 图片质量要求高
- ✅ 在高分辨率屏幕展示
- ✅ 内容特别丰富

**CLI 使用 4K**:
```bash
python main.py paper.pdf -t academic_panels_landscape.html --preset screen_4k
```

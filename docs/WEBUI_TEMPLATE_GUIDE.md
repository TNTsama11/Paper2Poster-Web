# Web UI 模板选择指南

## 📋 模板选择器位置

在两个 Web UI 中，模板选择器都在 **"🎨 高级功能"** 区域：

### Gradio UI
1. 打开 http://localhost:7860
2. 展开 **"🎨 高级功能"** 折叠面板
3. 找到 **"海报模板"** 单选按钮组
4. 选择你想要的模板

### Streamlit UI
1. 打开 http://localhost:8501
2. 在左侧边栏找到 **"🎨 高级功能"** 展开区域
3. 找到 **"海报模板"** 单选按钮组
4. 选择你想要的模板

---

## 🎨 三种模板详解

### 1️⃣ 经典三栏 (simple_grid)

**适用场景**:
- 🚀 快速测试和预览
- 📄 图片较少的论文（1-2张图）
- 🎯 注重内容文字的海报

**特点**:
- ✅ 三栏式布局，清晰简洁
- ✅ 单张主图展示
- ✅ 生成速度最快
- ✅ 不需要视觉分析
- ⚠️ 空间利用率较低（约 70%）

**推荐指数**: ⭐⭐⭐ (入门推荐)

---

### 2️⃣ 多图丰富 (multi_figure_grid) ⭐

**适用场景**:
- 📊 图片较多的论文（3-6张图）
- 💼 日常使用推荐
- 🎨 需要展示多个实验结果
- 🏆 商业展示

**特点**:
- ✅ 最多显示 6 张图片
- ✅ 智能图片放置
- ✅ 内容丰富饱满
- ✅ 空间利用率高（约 85%）
- ⚠️ **需要启用视觉分析**
- ⚠️ 生成速度适中

**推荐指数**: ⭐⭐⭐⭐ (日常推荐)

**配置要求**:
```
✅ 勾选 "启用视觉分析"
✅ 配置视觉模型 (或留空自动推荐)
```

---

### 3️⃣ 学术会议 (academic_panels) ⭐⭐⭐

**适用场景**:
- 🎓 学术会议海报展示
- 🏅 高质量论文展示
- 🔬 需要专业视觉效果
- 🎯 追求最佳布局

**特点**:
- ✅ 9 个独立面板布局
- ✅ 灵活的面板尺寸（支持 1x1, 1x2, 2x1, 2x2 等）
- ✅ 最高空间利用率（约 95%）
- ✅ 专业学术会议风格
- ✅ 参考 Paper2Poster 项目设计
- ⚠️ **需要启用视觉分析**
- ⚠️ 生成速度较慢

**推荐指数**: ⭐⭐⭐⭐⭐ (专业推荐)

**配置要求**:
```
✅ 勾选 "启用视觉分析"
✅ 配置视觉模型 (推荐 GPT-4V 或 Qwen-VL)
✅ 建议使用更高分辨率 (1200x1800)
```

## 🎯 快速选择建议

| 使用场景 | 推荐模板 | 理由 |
|---------|---------|------|
| 快速测试 | 经典三栏 | 速度快，不需视觉分析 |
| 日常使用 | 多图丰富 | 平衡美观与性能 |
| 学术会议 | 学术会议 | 最专业的效果 |
| 图少 (1-2张) | 经典三栏 | 避免空白浪费 |
| 图多 (3-6张) | 多图丰富 | 充分展示图片 |
| 图多 (>6张) | 学术会议 | 最大空间利用 |
| 没有视觉模型 API | 经典三栏 | 唯一不需视觉分析的模板 |

## 🔧 配置步骤

### 使用 "经典三栏" 模板
1. ✅ 上传 PDF
2. ✅ 配置 API Key 和模型
3. ✅ 选择 "经典三栏 (simple_grid)"
4. ✅ 点击生成
5. ⏱️ 等待 30-60 秒

### 使用 "多图丰富" 或 "学术会议" 模板
1. ✅ 上传 PDF
2. ✅ 配置 API Key 和主模型
3. ✅ **勾选 "启用视觉分析"**
4. ✅ 配置视觉模型 (或留空自动推荐)
   - 推荐: `gpt-4o-mini` (OpenAI)
   - 推荐: `Qwen/Qwen2-VL-7B-Instruct` (SiliconFlow)
5. ✅ 选择模板 "多图丰富" 或 "学术会议"
6. ✅ 点击生成
7. ⏱️ 等待 60-120 秒 (取决于图片数量)

## 💡 Pro Tips

### 提升生成质量
1. **图片多**: 优先选择 "学术会议" 模板
2. **时间紧**: 选择 "经典三栏" 快速预览
3. **求平衡**: "多图丰富" 是最佳日常选择
4. **高分辨率**: 学术会议模板建议使用 1200x1800 或更高

### 节省 API 成本
1. **测试阶段**: 使用 "经典三栏"（不调用视觉 API）
2. **正式生成**: 再使用 "学术会议" 生成最终版本
3. **批量生成**: 考虑使用本地视觉模型（如 Qwen-VL）

### 常见问题

**Q: 为什么选择 "多图丰富" 后还是只显示 1 张图？**  
A: 需要勾选 **"启用视觉分析"** 并配置视觉模型。

**Q: 视觉模型留空会怎样？**  
A: 系统会根据你的主模型 API 自动推荐：
   - OpenAI API → 使用 `gpt-4o-mini`
   - SiliconFlow API → 使用 `Qwen/Qwen2-VL-7B-Instruct`

**Q: 学术会议模板生成失败怎么办？**  
A: 
   1. 确认视觉分析已启用
   2. 确认视觉模型 API 配置正确
   3. 尝试降级到 "多图丰富" 模板
   4. 查看日志 `output/poster.log`

**Q: 可以自定义模板吗？**  
A: 
   - ✅ Web UI 暂不支持上传自定义模板
   - ✅ 使用命令行版本: `python main.py input.pdf -t my_template.html`
   - 📖 查看 [TEMPLATES_GUIDE.md](../TEMPLATES_GUIDE.md) 了解如何创建自定义模板

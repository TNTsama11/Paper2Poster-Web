# 快速参考：模板选择

## 🎯 在 Web UI 中选择模板

### Gradio (http://localhost:7860)
1. 展开 **"🎨 高级功能"**
2. 找到 **"海报模板"** 单选按钮
3. 选择你想要的模板

### Streamlit (http://localhost:8501)
1. 左侧边栏 → **"🎨 高级功能"**
2. 找到 **"海报模板"** 单选按钮
3. 选择你想要的模板

---

## 📋 三种模板快速对比

| 模板 | 推荐指数 | 图片数 | 需要视觉分析 | 生成时间 | 适用场景 |
|------|---------|-------|------------|---------|---------|
| **经典三栏** | ⭐⭐⭐ | 1张 | ❌ | 40秒 | 快速测试 |
| **多图丰富** | ⭐⭐⭐⭐ | 最多6张 | ✅ | 80秒 | 日常使用 |
| **学术会议** | ⭐⭐⭐⭐⭐ | 最多9张 | ✅ | 100秒 | 专业展示 |

---

## 💡 快速选择

**场景 1: 我只想快速测试**
→ 选择 **"经典三栏 (simple_grid)"**  
✅ 不需要配置视觉分析  
✅ 生成速度最快

**场景 2: 我需要展示多张图片**
→ 选择 **"多图丰富 (multi_figure_grid)"**  
✅ 勾选 "启用视觉分析"  
✅ 配置或留空视觉模型

**场景 3: 我要参加学术会议**
→ 选择 **"学术会议 (academic_panels) ⭐"**  
✅ 勾选 "启用视觉分析"  
✅ 推荐配置高质量视觉模型  
✅ 建议使用高分辨率 (1200x1800)

---

## ⚙️ 完整配置示例

### 使用 "学术会议" 模板 (推荐)

**Gradio / Streamlit 配置**:
```
PDF 文件: ✅ 上传你的论文
API Key: ✅ sk-xxxxx
API Base URL: ✅ https://api.siliconflow.cn/v1
模型名称: ✅ Qwen/Qwen2.5-7B-Instruct
输出格式: ✅ PDF (或 HTML/PNG)
海报宽度: ✅ 1200
海报高度: ✅ 1800

--- 🎨 高级功能 ---
启用视觉分析: ✅ 勾选
视觉模型: ✅ Qwen/Qwen2-VL-7B-Instruct (或留空)
海报模板: ✅ 学术会议 (academic_panels) ⭐
```

点击 **"生成海报"** → 等待 90-120 秒 → 完成！

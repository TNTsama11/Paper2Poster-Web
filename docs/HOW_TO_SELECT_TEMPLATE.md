# 如何在 Web UI 中选择模板？

## 🎯 快速答案

### 在 Gradio UI (http://localhost:7860)
1. 向下滚动到 **"🎨 高级功能"** 区域
2. 点击展开（如果已折叠）
3. 找到 **"海报模板"** 单选按钮组
4. 选择你想要的模板：
   - ○ 经典三栏 (simple_grid)
   - ● 多图丰富 (multi_figure_grid)  ← 默认
   - ○ 学术会议 (academic_panels) ⭐

### 在 Streamlit UI (http://localhost:8501)
1. 查看左侧边栏
2. 找到 **"🎨 高级功能"** 展开区域
3. 找到 **"海报模板"** 单选按钮组
4. 选择你想要的模板

---

## 🎨 三种模板简介

### 1. 经典三栏
- **特点**: 简洁清晰，单张主图
- **适合**: 快速测试，图片少的论文
- **视觉分析**: 不需要 ❌

### 2. 多图丰富 (推荐)
- **特点**: 最多6张图，内容丰富
- **适合**: 日常使用，图片较多的论文
- **视觉分析**: 需要 ✅

### 3. 学术会议 (最专业)
- **特点**: 9面板布局，空间利用率最高
- **适合**: 学术会议展示，追求最佳效果
- **视觉分析**: 需要 ✅

---

## ⚠️ 重要提示

**使用 "多图丰富" 或 "学术会议" 模板时**:
1. 必须勾选 ☑ **"启用视觉分析"**
2. 配置 **"视觉模型"** (或留空自动推荐)

否则只会显示 1 张图片！

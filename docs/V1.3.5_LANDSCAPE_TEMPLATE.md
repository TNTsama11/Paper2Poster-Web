# v1.3.5 横向16:9学术海报模板

## 新增功能

### 📺 学术会议横向模板 (academic_panels_landscape)

为了适应屏幕展示和在线会议，新增 **16:9 横向布局**的学术海报模板。

---

## 设计特点

### 1. 横向布局 (16:9)

**推荐尺寸**: `3840×2160` (4K) 或 `1920×1080` (1080p)

**布局结构**:
```
┌─────────────────────────────────────────────────────┐
│                  标题栏 (Header)                     │
├──────────────────────┬──────────────────────────────┤
│                      │                              │
│   左侧栏 (35%)       │   右侧栏 (65%)               │
│                      │                              │
│   - Abstract         │   ┌─────────┬─────────┐    │
│   - Key Figure       │   │  Intro  │ Methods │    │
│                      │   ├─────────┼─────────┤    │
│                      │   │ Results │Conclusion│   │
│                      │   ├─────────┴─────────┤    │
│                      │   │  Key Equations    │    │
│                      │   ├───────────────────┤    │
│                      │   │   References      │    │
│                      │   └───────────────────┘    │
└──────────────────────┴──────────────────────────────┘
│                  页脚 (Footer)                       │
└─────────────────────────────────────────────────────┘
```

### 2. 左右分栏设计

**左侧栏 (35%)**:
- **Abstract**: 完整显示摘要，无省略号
- **Key Figure**: 大尺寸主图展示

**右侧栏 (65%)**:
- **2×2网格**: Introduction, Methods, Results, Conclusion
- **宽面板**: Key Equations, References

### 3. 优化的显示效果

- ✅ **Abstract 完整显示**: 无截断、无省略号
- ✅ **大尺寸主图**: 最大高度 600px
- ✅ **平衡布局**: 左侧突出重点，右侧详细内容
- ✅ **适合投影**: 16:9 黄金比例

## 使用方法

### CLI 命令行

```bash
# 使用4K尺寸 (推荐)
python main.py paper.pdf -t academic_panels_landscape.html

# 自动使用 screen_4k 预设 (3840×2160)
```

### Web UI (Gradio)

1. 在"海报模板"选择：  
   **学术会议-横向16:9 (academic_panels_landscape) 🆕**

2. 尺寸会自动选择 `screen_4k` (3840×2160)

3. 或手动选择其他16:9尺寸：
   - `screen_hd` (1920×1080) - 轻量级
   - `screen_4k` (3840×2160) - 高清

## Abstract 显示优化

### 问题修复

**问题**: 原纵向模板中 Abstract 有时出现省略号 "..." 和文本截断

**原因**:
- Grid 布局 + 固定高度
- `overflow: hidden` 导致溢出隐藏

**横向模板解决方案**:
1. **独立左侧栏**: Abstract 不受网格限制
2. **自适应高度**: `height: fit-content`
3. **完整显示**: `overflow: visible`
4. **合理字号**: `font-size: 0.9rem`
5. **文本对齐**: `text-align: justify`

## 样式特点

### 左侧栏样式
```css
.left-column {
    flex: 0 0 35%;  /* 固定35%宽度 */
    display: flex;
    flex-direction: column;
    gap: 15px;
}

.panel-abstract {
    flex: 0 0 auto;  /* 自适应高度 */
    height: fit-content;  /* 完整显示 */
}

.panel-main-figure {
    flex: 1;  /* 占据剩余空间 */
}
```

### 右侧栏样式
```css
.right-column {
    flex: 1;  /* 占据剩余空间 */
    display: grid;
    grid-template-columns: repeat(2, 1fr);  /* 2列 */
    grid-auto-rows: minmax(150px, auto);
    gap: 12px;
}
```
